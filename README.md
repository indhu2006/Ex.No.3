### Aim:
To write the prompts for these following prompt types and evaluate that using any one method 1. Straightforward Prompts, 2. Tabular Format Prompting 3.Preceding Question Prompting and 4. Missing Word Prompting

### Explanation

Procedure:
Prompt engineering plays an important role in guiding large language models to generate relevant and accurate responses. Different types of prompts help in obtaining structured, detailed, or concise outputs depending on the requirement. In this report, we use the use case of Sentiment Analysis on Product Reviews (from Unit 5) and apply four different prompt types: Straightforward Prompts, Tabular Format Prompting, Preceding Question Prompting, and Missing Word Prompting. The outputs are then evaluated using the Accuracy and Completeness method.

Prompts and Outputs
1. Straightforward Prompts

Prompt: "Define sentiment analysis in one sentence."

Output: Sentiment analysis is the process of using natural language processing techniques to determine whether a text expresses a positive, negative, or neutral opinion.

Prompt: "List three real-world applications of sentiment analysis."

Output:

Analyzing customer reviews of products and services.

Monitoring social media sentiment about brands.

Understanding public opinion on political issues.


2. Tabular Format Prompting (Unit 2 Prompt Type)

Prompt: "Provide a table listing three sentiment analysis techniques, their approach, and one limitation each."

Output:

Technique	Approach	Limitation
Lexicon-based Analysis	Uses predefined dictionary of words	Struggles with sarcasm and context
Machine Learning Models	Trains classifiers on labeled datasets	Requires large annotated data
Deep Learning (LSTMs, BERT)	Uses neural networks to capture context	High computational cost


3. Preceding Question Prompting

Prompt: "Why is sentiment analysis important for businesses? Explain how Natural Language Processing (NLP) helps in analyzing customer reviews."

Output: Sentiment analysis helps businesses understand customer satisfaction, improve products, and enhance decision-making. NLP techniques process customer reviews by identifying keywords, context, and tone, thereby categorizing them into positive, negative, or neutral sentiments.


4. Missing Word Prompting

Prompt: "Sentiment analysis is the process of determining whether a piece of text expresses a ____ opinion."

Output: positive, negative, or neutral.

Prompt: "Customer reviews can be categorized into positive, negative, or ____ sentiments."

Output: neutral.

### Conclusion 
From the experiment, it is evident that different prompt types serve different purposes. Straightforward prompts are useful for quick definitions, Preceding Question prompts provide reasoning-based explanations, and Missing Word prompts help in factual recall. However, the Tabular Format Prompting was the most effective for the chosen use case, as it provided structured, detailed, and comparative information, making it the most suitable format for generating academic or professional reports.
